# Employee Record Management System

## Description
A Java-based console application to manage employee records using Java Collections.

## Features
- Add new employee
- Update employee details
- Delete employee records
- View all employees
- Search employee by ID

## Technologies Used
- Java
- ArrayList
- HashMap
- HashSet
- Scanner class

## Purpose
This project was developed as part of my Java internship to demonstrate practical usage of Java Collections.
